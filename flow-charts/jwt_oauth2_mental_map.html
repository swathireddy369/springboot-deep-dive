üîÑ Side-by-Side Flows

JWT Authentication Flow

User Login
Client sends credentials to auth server
JWT Creation
Server creates JWT with user claims + signature
Token Return
JWT sent back to client (cookie/localStorage)
API Calls
Client includes JWT in Authorization header
Token Validation
Server verifies signature + expiry, grants access

OAuth2 Authorization Code Flow

Authorization Request
Client redirects user to auth server with client_id & scopes
User Consent
User logs in and approves permissions
Auth Code Return
Auth server redirects back with authorization code
Token Exchange
Client exchanges auth code + client_secret for access token
API Access
Client uses access token to call protected APIs
